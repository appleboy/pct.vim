=======
pct.vim
=======

----------
How to use
----------

1. clone to your home dir::

    $ git clone git://github.com/pct/pct.vim.git

2. update submodule in pct.vim::

    $ cd pct.vim; ./update_submodule.sh

3. link .vimrc and .vim from pct.vim::

    $ ln -s pct.vim/.vimrc
    $ ln -s pct.vim/.vim 

-------------
How to update
-------------
::

    $ cd pct.vim
    $ git pull; ./update_submodule.sh
